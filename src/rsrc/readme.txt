 +----------------------------------------------------------------------+
 |      Screensaver Demo Loader for "1995" Demo by kewlers and mfx      |
 +----------------------------------------------------------------------+

Version...: v1.0
Date......: 19 Oct 2007


I) What's that
==============

This simple screensaver will convert the "1995" demo by kewlers and mfx to be usable as
a "native" Windows screensaver.

It supports somes options (resolution, screen format, sound and FSAA).

II) How to use it
=================

Download the "1995" original demo: http://pouet.net/prod.php?which=25783
Unzip it in a 1995 directory, at the same level of the "scrnload.scr" proggy.

You should have now:
--\
  |_ scrnload.scr
  |_ 1995\
         |_test3d.exe
         |_ <some others files>

Then after doing this, you can move the scrnload.scr and the 1995 folder in your system32
directory for example. Then, right-click on the scrnload.scr and select "Install". You can
configure the Screensaver Demo Loader as you want.

PS: For Win9x users, you must move the scrnload.scr and the 1995 demo in your windows\system
directory. Also, if you want a friendly name for the screensaver, rename the scrnload.scr as
"1995 screensaver.scr" for example. Password screensaver protection should work too.
